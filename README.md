# cafe-portal

## Project setup
```
npm install
```

### config files
you can configure the project by edit `config/config.json`


### Compiles and hot-reloads for development
```
npm run serve:local
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Serve APP at Prod environment
```
npm run serve
```

### Lints and fixes files
```
npm run lint
```
